## 免签约个人微信/支付宝收款码支付系统

### 系统包含的组件
- 支付管理
  - Golang后端程序
  - Vue前端程序
  - Mysql
- Android apk安装包

### 说明
#### 支付管理
```
已经打包成了docker镜像和二进制程序(支持linux/win/mac)
用于记录管理订单，个人支付码，支付成功回调参数，登录日志，Android APK回调日志，以及管理账号设置
```

#### Android程序
```
已经打包成apk了，用于发送回调信息以及显示支付日志

```
